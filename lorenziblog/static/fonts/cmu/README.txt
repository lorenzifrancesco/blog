Drop CMU font files in this folder and keep the filenames in sync with
lorenziblog/assets/css/custom.css.

Expected files (woff2):
- CMUSerif-Roman.woff2
- CMUSerif-Italic.woff2
- CMUSerif-Bold.woff2
- CMUSerif-BoldItalic.woff2
- CMUTypewriter-Regular.woff2
- CMUTypewriter-Italic.woff2
- CMUTypewriter-Bold.woff2
- CMUTypewriter-BoldItalic.woff2

If your files use different names, either rename them here or update the
@font-face src paths in custom.css to match.
